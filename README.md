# 🎂 Lulu猪的3D旋转相册

> 一个精美的3D生日相册，带有弹幕祝福、音乐播放和丰富的动画效果。

---

## 🌟 为什么选择这个相册？

- ✅ **精美3D效果** - 双层立方体结构，12个相框展示

- ✅ **交互丰富** - 支持拖拽旋转、呼吸式展开动画

- ✅ **个性化祝福** - 输入名字即可生成专属弹幕祝福

- ✅ **氛围感十足** - 爱心雨、表情漂浮、背景音乐

- ✅ **完全免费** - 纯前端实现，无需服务器

## 🚀 3 步快速上手

### 第一步：准备文件

确保项目文件夹中包含以下文件：
- `index.html` - 主页面
- `style.css` - 样式文件
- `script.js` - 逻辑文件
- `生日快乐.mp3` - 生日歌音频（可选）

### 第二步：替换照片（可选）

打开 `index.html` 文件，找到图片标签并替换为你的照片：

```html
<!-- 外部立方体照片 -->
<img src="img/photo1.jpg" class="out_pic" alt="照片">

<!-- 内部立方体照片 -->
<img src="img/photo2.jpg" class="in_pic" alt="照片">
```

> 💡 **提示**：
> - 将你的照片文件放在 `img/` 文件夹中，然后修改 `src` 属性中的文件名即可
> - 建议使用正方形照片（400x400或200x200），效果最佳

### 第三步：开始使用

**方式一：直接打开（推荐）**

直接用浏览器打开 `index.html` 文件即可使用。

**方式二：使用本地服务器（可选）**

如果直接打开遇到问题，可以使用本地服务器：

```bash
# 使用 Node.js（无需安装，直接运行）
npx http-server

# 或使用 Python（如果已安装）
python -m http.server 8000
```

> 💡 **提示**：大多数情况下直接打开 `index.html` 即可正常使用，无需启动服务器。

## 📱 如何使用

```
1️⃣ 输入名字  →  2️⃣ 点击开始祝福  →  3️⃣ 享受动画和音乐
```

### 详细步骤

1. **输入寿星名字**
   - 页面加载后，输入框自动获得焦点
   - 输入寿星的名字（最多10个字符）

2. **开始祝福**
   - 点击"🎵 开始祝福 🎵"按钮
   - 音乐开始播放，弹幕祝福出现
   - 相框开始呼吸式展开和收缩

3. **交互体验**
   - 可以拖拽相册旋转查看不同角度
   - 鼠标悬停时相框会展开
   - 欣赏爱心雨和表情漂浮特效

## 🔧 自定义配置

### 修改祝福语

在 `script.js` 中找到 `birthdayWishes` 数组，可以添加或修改祝福语：

```javascript
const birthdayWishes = [
    "🎂 {name}，生日快乐！愿你天天开心！",
    "🎉 {name}，祝你生日快乐，身体健康！",
    // 添加更多祝福语...
];
```

### 修改动画速度

在 `style.css` 中修改动画时长：

```css
#react.playing .out_front {
    animation: breatheExpand 3s cubic-bezier(0.4, 0, 0.6, 1) infinite;
    /* 修改 3s 来改变速度 */
}
```

### 修改背景颜色

在 `style.css` 中修改 `body` 的背景渐变：

```css
body {
    background: linear-gradient(135deg, #ffb3d9 0%, #ff99cc 25%, ...);
}
```

### 修改文件大小限制

在 `script.js` 中找到相关常量进行修改：

```javascript
// 例如：修改输入框最大长度
maxlength="10"  // 在 index.html 中修改
```

## ❓ 常见问题

**Q: 音乐无法播放怎么办？**  
A: 检查 `生日快乐.mp3` 文件是否在项目根目录，确保浏览器支持音频播放。

**Q: 可以添加更多照片吗？**  
A: 可以，在 `index.html` 中修改图片路径，建议使用正方形照片效果最佳。

**Q: 弹幕祝福可以自定义吗？**  
A: 可以，在 `script.js` 中的 `birthdayWishes` 数组添加或修改祝福语。

**Q: 支持哪些浏览器？**  
A: 支持 Chrome、Firefox、Safari、Edge 等现代浏览器，需要支持 CSS 3D Transform。

**Q: 如何修改相册旋转速度？**  
A: 在 `style.css` 中找到 `animation: rotate 20s infinite;`，修改 `20s` 即可。

**Q: 可以关闭某些特效吗？**  
A: 可以，在 `script.js` 中注释掉相应的函数调用即可。
